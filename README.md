# blockliss.github.io
🧠 Blockly JS Game & Website Builder - Help Guide

---

👋 Welcome

This program lets you build JavaScript programs, games, and websites using blocks instead of typing code.

If you’ve never used JavaScript before, don’t worry — just drag blocks and the program creates the code for you.

---

🧭 Layout Explained

🧩 Left Side (Block Area)

- Drag blocks from the toolbox
- Snap them together like puzzle pieces

🖥️ Right Side

- Run Code → Runs your program
- Clear Output → Clears the result
- Export JS → Downloads your code as a ".js" file
- Export HTML → Downloads a full website

📜 Generated JS

- Shows the real JavaScript your blocks create

🌐 Live Output

- Shows what your program does

---

🚀 Basic How To Use

1. Drag blocks from the toolbox
2. Connect them together
3. Click Run Code
4. Watch the result

---

🧱 Important Block Categories

🧠 Logic

- If / else blocks
- Used for decisions

🔁 Loops

- Repeat actions multiple times

➗ Math

- Numbers and calculations

🔤 Text

- Words and messages

📦 Variables

- Store values (like score, name, etc.)

📚 Lists

- Store multiple items (like inventory)

🌐 Game / Web

- Create elements (boxes, images)
- Move objects
- Handle clicks
- Show alerts

---

📦 Variables (VERY IMPORTANT)

Variables store data.

Example:

- Set variable "score" to "0"
- Change variable "score" by "1"

Used for:

- Score systems
- Counters
- Game stats

---

📚 Lists (Arrays)

Lists store multiple values.

Example:

- Create list "items"
- Add ""apple"" to list
- Add ""banana"" to list

Used for:

- Inventories
- Enemy lists
- Data storage

---

⚠️ Alert Block

The alert block shows a popup message.

Example:

alert "Hello!"

Use it to:

- Show messages
- Debug your program

---

🖼️ Image Importing

The import image block:

- Opens a file picker
- Lets you choose an image
- Adds it to your page

Good for:

- Games
- Visual apps

---

🖱️ Events (Interaction)

Example:

When box clicked → do something

Use this to:

- Make buttons
- Create games
- Add interactivity

---

⏱️ Timers

Wait blocks allow delays.

Example:

Wait 1000 ms → do something

👉 1000 ms = 1 second

---

💾 Exporting Your Project

Export JS

- Saves your code as a JavaScript file

Export HTML

- Saves a full working website

👉 Tip: Use HTML export to share your project easily

---

💡 Beginner Tips

1. Start simple
   → Make a box and change its text

2. Use console log or alert to debug

3. Build step by step
   → Don’t try to make a full game at once

4. If something breaks:
   
   - Check block connections
   - Run smaller parts first

---

🎮 Example First Project

1. Create element ("div") with id "box"
2. Set text of "box" to ""Click me!""
3. When "box" clicked:
   → Alert ""You clicked it!""

---

🧠 Final Note

You are learning real JavaScript — the blocks are just a visual way to write it.

As you improve, you can:

- Read the generated code
- Understand how JavaScript works
- Eventually write code yourself
